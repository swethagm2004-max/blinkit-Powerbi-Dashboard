1. **Data Collection**
   The Blinkit dataset containing sales, outlet details, product categories, and performance metrics was collected.

2. **Data Cleaning**
   Data was cleaned using Power Query by handling missing values, correcting data types, and removing inconsistencies.

3. **Data Transformation**
   Required columns such as total sales, average sales, and category-wise metrics were prepared for analysis.

4. **Data Modeling**
   Relationships between tables were created to ensure accurate reporting and analysis.

5. **DAX Calculations**
   Important KPIs like Total Sales, Average Sales, Number of Items, and Outlet Performance were calculated using DAX formulas.

6. **Dashboard Design**
   Interactive charts, cards, slicers, and filters were designed to present sales trends, outlet analysis, and category performance clearly.

7. **Analysis & Insights**
   The dashboard was used to identify sales trends, top-performing outlets, and high-demand product categories.
